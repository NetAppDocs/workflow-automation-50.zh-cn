= 在 Active IQ 数据中心管理器上配置数据库用户
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须在 Active IQ Data Center Manager 6.0 之前的版本上创建一个数据库用户，才能配置对 OnCommand Workflow Automation 的 Active IQ Data Center Manager 数据库的只读访问权限。
