== WFA 版本信息

* * 版本 *
+
显示已安装 WFA 应用程序的当前版本号。

+
版本号的格式如下： `A.B.C` 。

+
`A.B.C` 反映了 _MAJER.Minal.Maintention_ 版本号。

+
例如： `2.0.0` 。

* * 系统 ID*
+
标识 WFA 系统安装，以便进行跟踪和使用 AutoSupport 。

+
这是系统生成的唯一标识符。
